## File Description
TLT_Combiner_S21_Simulation_Data.xlsx  
Simulation data exported from QucsStudio.  
Frequency range: 45–870 MHz (VHF–UHF band)

## Worksheets
- FT37 S21.csv — S21 response data for FT37-61 core
- FT37.csv — Additional FT37-61 simulation output
- FT50 S21.csv — S21 response data for FT50-61 core
- FT50.csv — Additional FT50-61 simulation output
- 4port S21.csv — S21 response data for 4-way combiner
- 4port.csv — Additional 4-port combiner simulation output
